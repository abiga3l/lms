Introduction to HTML

Introduction to HTML is designed to provide learners with a comprehensive understanding of Hypertext Markup Language (HTML). Throughout the course, students will embark on a journey into the world of web development, starting with the basics of HTML syntax, tags, and elements. From structuring content to creating links and embedding media, learners will gain hands-on experience in crafting well-formed HTML documents. There will be practical exercises and demonstrations. The course has quizzes and assignments where the learner needs to complete to see their progress and understanding. Throughout the course, emphasis will be placed on fostering a supportive learning environment where learners feel empowered to experiment and ask questions, hence expanding their HTML proficiency at their own pace.

The instructor for this course is MR. GERALD

CSS Fundamentals

The CSS Fundamentals is designed to equip learners with the essential skills needed to style and design modern web pages using Cascading Style Sheets (CSS). The learners will learn how to apply styles to HTML elements to control aspects such as layout, typography, colors, and effects. Through hands-on exercises and projects, learners will gain practical experience in creating visually appealing and responsive web designs. By the end of the course, participants will have the proficiency to transform basic HTML documents into professionally styled web pages, setting the foundation for advanced front-end development skills.

The instructor is MRS. JOY

JavaScript Basics

The JavaScript Basics course offers a comprehensive introduction to the fundamental concepts of the JavaScript programming language. Participants will explore the core syntax, data types, variables, and operators essential for building interactive and dynamic web applications. Through practical examples and exercises, learners will understand how to manipulate the Document Object Model (DOM) to dynamically update and modify HTML and CSS content. Additionally, the course will cover essential programming concepts such as control structures (e.g., loops, conditionals), functions, and event handling to enhance interactivity and user experience. By mastering the basics of JavaScript, participants will lay a solid foundation for further exploration into front-end and full-stack web development.

The instructor is MS. MACHARIA

Web Project Management

The Web Project Management course provides learners with the knowledge and skills necessary to effectively plan, execute, and oversee web development projects. Participants will learn about project management methodologies, tools, and techniques tailored specifically for the dynamic environment of web development. Topics covered may include project initiation, scope definition, task scheduling, resource allocation, risk management, and quality assurance. Through case studies and real-world scenarios, learners will gain insights into managing various aspects of web projects, including collaboration with cross-functional teams, client communication, and stakeholder engagement. By the end of the course, participants will be equipped with the strategies and best practices needed to lead successful web development projects from inception to completion.

The instructor is MR. WEKESA
